# Pécory Coulibaly – Blog

Site personnel de Pécory Coulibaly, journaliste et chercheure.  
Hébergé gratuitement via [GitHub Pages](https://pages.github.com).

## Installation rapide

1. Crée un dépôt `username.github.io` (ou utilise n’importe quel nom pour une page de projet).
2. Dépose tous les fichiers à la racine.
3. Va dans *Settings &gt; Pages* et sélectionne **Deploy from a branch &gt; main**.
4. Remplace `images/pecory.jpg` par la photo réelle (garder le même nom).

## Ajouter un nouvel article

1. Crée un nouveau fichier HTML dans la racine : `article-mon-titre.html`.
2. Copie-colle la structure de `article-ia-banques-cote-ivoire.html`.
3. Mets à jour `index.html` en ajoutant un nouveau bloc `.card` dans la section `.articles`.
